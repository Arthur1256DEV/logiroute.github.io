🚚 LogiRoute | Otimizador Inteligente de Entregas
O LogiRoute é uma solução web desenvolvida para simplificar e otimizar o planejamento de rotas de entregas urbanas. Combinando geocodificação, roteamento avançado e tecnologias modernas de captura de dados, o sistema permite que entregadores e logísticos planejem seus percursos de forma eficiente, reduzindo tempo e distância.

🚀 Funcionalidades Principais
Otimização Inteligente (VROOM): Utiliza algoritmos de otimização de rota (Problema do Caixeiro Viajante) para encontrar a sequência mais eficiente de paradas.

Gestão Dinâmica de Rota: Adicione, remova ou reordene paradas através de uma interface intuitiva com suporte a Drag and Drop.

Leitura de QR Code Nativa: Integração com a câmera do dispositivo para bipar etiquetas de entrega, suportando tanto texto simples quanto objetos JSON estruturados.

Visualização Interativa: Mapa dinâmico integrado com Leaflet e estilo "Dark Mode" para melhor legibilidade.

Integração com Google Maps: Botão de exportação que envia a sequência otimizada diretamente para a navegação do Google Maps no smartphone do entregador.

🛠️ Tecnologias Utilizadas
Frontend: HTML5, CSS3, JavaScript (Vanilla).

Mapas: Leaflet.js com tiles do CartoDB.

Roteamento e Otimização: OpenRouteService API.

Funcionalidades Extras:

SortableJS para ordenação manual de itens.

html5-qrcode para leitura de códigos via câmera.

📦 Como funciona a Otimização
O LogiRoute trata o problema de entregas de duas formas:

Modo Manual: Permite que o operador ajuste a ordem das paradas manualmente baseando-se em restrições locais (drag and drop).

Modo Inteligente: Utiliza a API de otimização para calcular a menor rota possível entre todos os pontos, automatizando a tomada de decisão logística.
